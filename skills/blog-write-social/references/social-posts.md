# Social Posts

## LinkedIn (Knowledge Card Format)
- Start with emoji + chapter reference:
  - Deep Learning topics: ∇ **Deep Learning Book (Chapter X.X)**
  - Linear Algebra topics: 📐 **MIT 18.06SC Linear Algebra (Lecture XX)**
  - Probability topics: 𝐏 **MIT 6.041 Probability (Lecture X)**
- Add a clear statement or equation line.
- 2-4 bullets with 🔹 for key insights.
- Philosophical closing line with 💡.
- Link line with 📖.
- Source attribution at the end.
- Include hashtags: #MachineLearning #LinearAlgebra #DeepLearning (trim to relevant ones if needed).
- Tone: "connecting the dots" (link concepts to intuition or applications).
- Default base URL for blog links: https://ickma2311.github.io/

Template:
∇ **Deep Learning Book (Chapter X.X)**
Learning ≠ Optimization
🔹 ...
🔹 ...
💡 ...
📖 <link>
My notes on Deep Learning (Ian Goodfellow) Chapter X.X
#MachineLearning #DeepLearning

## Twitter (Single Tweet)
- Single tweet only (no thread), <= 280 characters.
- 1-2 sentences with the key insight.
- Include the blog link.
- Use 1-2 relevant hashtags (e.g., #MachineLearning or #LinearAlgebra).
- No emoji unless the user asks.
- Before drafting, suggest 5-10 search keywords/phrases to find hot tweets on the topic.
- State that you cannot access Twitter; provide keywords + a short reply angle the user can use.

Template:
Key insight in one line. Short context. <link> #MachineLearning
